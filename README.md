## ⚙️ Instalación

Clonar el repositorio

```bash
git clone https://github.com/usuario/repositorio.git
```

Entrar al proyecto

```bash
cd repositorio
```

Crear entorno virtual

```bash
python -m venv venv
```

Activarlo

Windows

```bash
venv\Scripts\activate
```

Linux / macOS

```bash
source venv/bin/activate
```

Instalar dependencias

```bash
pip install -r requirements.txt
```
